# React form

A small website - working with forms in React

## Run Locally

Clone the project

```bash
  git clone git@github.com:EidmantasDev/react-form.git
```

Go to the project directory

```bash
  cd react-form
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```
